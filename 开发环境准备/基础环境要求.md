# 基础环境要求





## 基础开发环境

- JDK: 1.8+ (小于11)
- Maven: 3.5+
- MySql: 5.7+
- Redis: 3.2 +
- Node Js: 10.0 +
- Npm: 5.6.0+
- Yarn: 1.21.1+
